# Bank-Management-System
BankManagementSystem is a console-based application developed using Core Java that simulates real-world banking operations. The project implements OOP principles, HashMap for account management, and custom exception handling to ensure secure and error-free transactions. It includes robust input validation to handle invalid user inputs gracefully without crashing the application.
